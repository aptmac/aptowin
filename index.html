This application was created and developed for entry into the Riot Games API Challenge 2.0 that took place between August 10th and August 13th of 2015. This application was developed for entry into category 2, which reads as follows:


Given two data sets of match IDs, one before the AP Item changes and another one after, create a piece of software to demonstrate changes in AP Item usage.


The supplied data sets contained tens of thousands of match identification numbers that I used to retrieve match data. I built scripts, using the Perl programming language, that downloaded all of the ranked match data (while adhering to my rate limit of 10 requests every 10 seconds), and read through all of the data to extract the prevalence of AP items used in the different roles (top, jungle, mid, adc, and support) among the different ranked tiers (unranked, bronze, silver, gold, platinum, diamond, master, challenger). For my own curiosity, I also pulled the numbers related to Teemo appearances as well (to see if the AP items made him a more favourable pick to the eyes of the world!).

All of the data I extracted is stored in JSON files, sorted by patch number, server region, and whether or not the items were used on the winning or losing team. All my code and data files are uploaded to my GitHub account (link below), so feel free to give it a look over and shoot me a message if you've got any questions, comments, or concerns.
